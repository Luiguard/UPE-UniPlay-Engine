SDK Reference

Universal API

One API to rule every genre. Plug your game logic into any module via standardized hooks and callbacks.

Register any game object – block, item, NPC, vehicle – with a unified type-safe registry.
TypeScript
import { UniPlayEngine } from '@uniplay/core';

// Register a custom block type
UniPlayEngine.RegisterObject('block', {
  id: 'diamond_ore',
  hardness: 5.0,
  dropTable: [{ item: 'diamond', chance: 0.8 }],
  physics: { solid: true, mass: 1200 },
  modules: ['WorldSim', 'Physics']
});

// Register a weapon
UniPlayEngine.RegisterObject('weapon', {
  id: 'ak47',
  damage: 34,
  fireRate: 600,
  recoilPattern: 'ak_recoil_v2',
  modules: ['Combat', 'Physics']
});
Zone callbacks fire when players enter, update, or leave map shards. Use them to load/unload simulation data dynamically.
TypeScript
// Zone lifecycle hooks
UniPlayEngine.OnZoneEnter((zoneID, player) => {
  console.log(`Player entered zone: ${zoneID}`);
  WorldSim.loadChunks(zoneID);
  NPC.spawnHorde(zoneID, 'wolf', { count: 5 });
});

UniPlayEngine.OnZoneUpdate((zoneID, delta) => {
  // Called every server tick for active zones
  Physics.step(zoneID, delta);
  Survival.tickEnvironment(zoneID, delta);
});

UniPlayEngine.OnZoneLeave((zoneID, player) => {
  WorldSim.unloadChunks(zoneID);
});
Hook into the simulation loop at any point. Each module exposes typed callback signatures.
TypeScript
// Physics step — called per tick per zone
Physics.OnPhysicsStep((ctx) => {
  ctx.bodies.forEach(body => {
    if (body.type === 'projectile') {
      Ballistics.advanceTrajectory(body, ctx.delta);
    }
  });
});

// Voxel change — fires when a block is modified
WorldSim.OnVoxelChange((voxel, action) => {
  if (action === 'break') {
    Lighting.recalculate(voxel.position, 16);
    Fluids.propagate(voxel.position);
  }
});

// NPC update — runs on assigned client cluster
NPC.OnNPCUpdate((npc, delta) => {
  BehaviorTree.tick(npc, delta);
  Pathfinding.step(npc);
});
Client-computed results flow through the Consensus Engine. Submit results and react to the validated truth.
TypeScript
// Each client submits their computed result
const taskID = 'hit_detection_tick_1042';

Consensus.SubmitResult(taskID, {
  hitPlayerID: 'player_77',
  damage: 34,
  hitzone: 'torso',
  timestamp: Date.now()
});

// React when the server has resolved consensus
Consensus.OnConsensusResolved(taskID, (finalResult) => {
  if (finalResult.hitPlayerID) {
    Health.applyDamage(finalResult.hitPlayerID, finalResult.damage);
    UI.showHitmarker(finalResult.hitzone);
  }
});
A universal pub/sub event bus for cross-module communication and game-specific logic.
TypeScript
// Send an event to all subscribers
EventBus.SendEvent('player.died', {
  playerID: 'player_12',
  cause: 'starvation',
  position: { x: 120, y: 64, z: -45 }
});

// Listen for events anywhere in your game logic
EventBus.OnEvent('player.died', (payload) => {
  Respawn.scheduleRespawn(payload.playerID, 30);
  UI.showDeathScreen(payload.cause);
  Analytics.record('death', payload);
});

// Wildcard subscriptions
EventBus.OnEvent('player.*', (type, payload) => {
  Telemetry.emit(type, payload);
});

👥
Client Cluster

Every player is a Worker Node. Computation is distributed across the player pool automatically by the Task Scheduler.

🔁
Redundancy

Each task is computed by 2–3 independent clients. Outliers are rejected. No single point of failure.

⚖️
Consensus

The server only arbitrates which result is valid. It never computes the result itself. Maximum efficiency.

State Anchor

The one central, untouchable truth that only the server may write to. The final 1–2% glitch eliminator.

SDK Integration

TypeScript · Server-Side Only
// Only the server may call StateAnchor.write()
// This is enforced at the transport layer — client calls are rejected

StateAnchor.write(entityID, {
  position:    consensus.position,    // Vec3 — from consensus winner
  velocity:    consensus.velocity,    // Vec3 — from consensus winner
  objectState: consensus.state,       // 'alive' | 'dead' | 'interacting'
  zoneMember:  consensus.zoneID,      // authoritative zone assignment
  playerFlags: consensus.flags,       // bitmask: grounded, sprinting…
  tick:        currentTick,           // monotonic tick counter
});

// Anchor is immediately broadcast to all zone subscribers
StateAnchor.broadcast(entityID, { zone: consensus.zoneID });
TypeScript · Client-Side
// Clients receive anchor updates and lerp smoothly
StateAnchor.OnAnchorUpdate((entityID, anchor) => {
  const entity = World.getEntity(entityID);

  // Smooth interpolation — never snap, always lerp
  entity.position.lerpTo(anchor.position, RECONCILE_SPEED);
  entity.velocity.set(anchor.velocity);

  // Hard-correct state (zone, flags) — these are boolean, no lerp
  entity.zone   = anchor.zoneMember;
  entity.flags  = anchor.playerFlags;
  entity.state  = anchor.objectState;

  // Rewind local prediction to match anchor tick
  Prediction.rewindTo(anchor.tick);
  Prediction.replayInputsSince(anchor.tick);
});
TypeScript · Server Hook
// Server-side validation hook — runs before any Anchor write
StateAnchor.OnBeforeWrite((entityID, proposed, current) => {

  // Plausibility: max movement per tick (anti-teleport)
  const delta = proposed.position.distanceTo(current.position);
  if (delta > MAX_MOVEMENT_PER_TICK) {
    return { accept: false, reason: 'position_teleport' };
  }

  // Zone boundary: only accept if correct zone transition
  if (proposed.zoneMember !== current.zoneMember) {
    if (!ZoneGraph.isAdjacent(current.zoneMember, proposed.zoneMember)) {
      return { accept: false, reason: 'invalid_zone_jump' };
    }
  }

  return { accept: true };
});

Production Ready

Enterprise Infrastructure

Built for millions of CCU. UniPlay Engine comes out-of-the-box with the operational backbone required for AAA production and massive horizontal scaling.

🔒

Zero-Trust Security

Strict packet sequence verification prevents Replay Attacks, while ECDSA cryptographic signatures guarantee that no node can forge consensus votes or state hashes.

🧮

Absolute Determinism

Native Fixed-Point Math library ensures calculations on ARM mobile chips and x86 desktop processors yield identical bits. Eradicates false-positive cheating bans caused by floating-point differences.

⚡

High-Availability (Redis)

The State Anchor periodically syncs memory to RedisJSON clusters. Tolerates hard Kubernetes Pod evictions scaling—spin up new edge nodes with zero data loss or client disconnections.

🧬

Binary UDP Serialization

Swapped bulky JSON for continuous flat binary buffers. Delivered over WebRTC Data Channels / UDP to eliminate TCP Head-of-Line blocking resulting in absolute minimal latency jitter.

📊

OpenTelemetry Observability

Full metrics registry exposing Prometheus-compatible endpoints. Monitor Quorum health, task distribution, ping latency variance, and anti-cheat strikes live on Grafana.

🔌

Universal Proxy Adapters

Includes Man-in-the-Middle Network Proxies and DMA Memory Hook adapters for older non-moddable games. Bring modern Distributed Consensus to 10-year-old single-player titles.
